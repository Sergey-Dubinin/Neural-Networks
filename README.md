## 🎯 О курсе

Привет! Этот репозиторий — мое видение идеального пути в Deep Learning. Здесь нет «черных ящиков». Мы начнем с линейной алгебры и матана, напишем свои первые нейросети на чистом NumPy, разберемся, как на самом деле работает Backpropagation, а затем перейдем к современным архитектурам, включая сверточные сети и механизмы внимания (Attention).

**Для кого этот курс?**
* 🚀 **Новичков**, которые хотят не просто импортировать `PyTorch`, а понимать, как всё устроено под капотом.
* 🛠 **Junior/Middle ML-инженеров**, которым нужно систематизировать знания и подтянуть математическую базу.
* 🎓 **Студентов**, готовящихся к собеседованиям в AI-сфере.

## 🚀 Как запустить локально

Если ты хочешь запускать ноутбуки у себя на компьютере, выполни следующие шаги:

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/Sergey-Dubinin/Neural-Networks.git
   cd Neural-Networks
2. Создай виртуальное окружение и активируй его:
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для Windows: venv\Scripts\activate
3. Установи зависимости:
    ```bash
    pip install -r requirements.txt
4. Запусти Jupyter Notebook:
    ```bash
    jupyter notebook

> ⚠️ Важное примечание для Урока 30
Для выполнения ноутбука 30_Encoder_and_Visualization.ipynb (визуализация механизмов внимания) требуется установленная в вашей операционной системе утилита FFmpeg.

**Как установить FFmpeg:**
* 🪟 Windows: Откройте терминал и выполните winget install ffmpeg.
* 🍎 macOS: Выполните в терминале brew install ffmpeg
* 🐧 Linux (Ubuntu/Debian): Выполните sudo apt update && sudo apt install ffmpeg

💡 Лайфхак: Если вы не хотите устанавливать FFmpeg на свой компьютер, просто откройте этот ноутбук через кнопку Open in Colab в таблице выше. В среде Google Colab FFmpeg уже предустановлен и всё заработает сразу!
